Programacion Orientado a Objetos

Es un paradigma de la programacion que actualiza la forma de programar anterior

Alguns de los conceptos fundamentales son:

  • Clase
  • Herencia
  • Objeto
  • Metodo
  • Evento
  • ETC

Clase

Son plantilla que se crean paracrear objetos iguales

Cuando creamos un bjeto a esa accion se le denomina INSTANCIAR un objeto

Necesitamos una funcion constructora. Se lellama constructor y se ejecuta cada vez que creemos el objeto

            class Persona {
                constructor(nombre, apellido, aded) {
                    this.nombre=nombre,
                    this.apellido=apellido,
                    this.edad=edad
                }
            }

Metodos

Una funcion pertenece a la clase se llama metodo, que es el comportamiento que va tener el objeto instanciado
