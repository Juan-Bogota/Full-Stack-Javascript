DOM

Trabajar HTML atravez de JavaScript

Es todo la estructura HTML del documento, No es JavaScript es una API(Aplication Programing Interface)

Cada parte del arbol del documento es un NODO

Hay varios tipos de Nodo, los mas utilizados son:

  • Elemento node - 1(CUalquier etiqueta HTML)
  • Text node - 3 (COntenido de la etiqueta)
  • Comment node - 8 (Cualquier comentario en HTML)
